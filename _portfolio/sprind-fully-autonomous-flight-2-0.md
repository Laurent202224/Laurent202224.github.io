In this project, we developed a fully autonomous hexacopter from scratch capable of performing last-mile delivery and search-and-rescue missions in unknown, GNSS-denied environments. The system was designed to operate robustly under real-world constraints, combining perception, planning, and control into a fully integrated autonomy stack.

On the hardware side, we designed and implemented the full system architecture. The platform was built around a Jetson AGX Orin (64GB) for onboard computation and a Pixhawk-based flight controller for low-level control. The sensor suite included three radars, dual forward-facing cameras, a downward-facing optical flow and laser sensor, and an external IMU. We developed the complete electrical integration, ensuring reliable communication between all components and enabling synchronized sensor data acquisition.

On the software side, we implemented a full autonomy pipeline:

- Navigation (SLAM): We used [OKVIS2-X](https://github.com/ethz-mrl/OKVIS2-X) as the core visual-inertial SLAM framework, adapting it to our multi-camera setup and enhancing robustness by fusing radar-based velocity estimates.
- Planning: We implemented an informative RRT* planner for efficient exploration of unknown environments. In addition, I developed several custom planners, including:
  - a go-to planner for visually guided target approach,
  - a local search planner for safe area coverage under uncertainty,
  - and a person-following planner for dynamic target tracking.

All trajectories generated by these global planners were executed using a 3D Dynamic Window Approach (DWA) local planner, optimizing velocities based on system dynamics and task-specific reward functions.

- Detection: For the delivery task, we trained a custom house number detector to localize target locations, followed by a vision-language model (Qwen VLM) to verify semantic task constraints. For search-and-rescue, we used YOLO-based human detection combined with VLM-based filtering to identify the correct person.

My primary contributions focused on the development of the planning stack and the mission manager coordinating the overall system behavior. I was also responsible for interfacing with the Pixhawk flight controller, ensuring seamless integration between high-level autonomy and low-level control.

The project successfully advanced to the second stage of the [SPRIND Fully Autonomous Flight 2.0](https://www.sprind.org/en/actions/challenges/funke-fully-autonomous-flight-2.0) challenge and was awarded €325k in funding, highlighting the system’s performance and innovation.
