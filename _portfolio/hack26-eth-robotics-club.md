During this 48-hour hackathon organized by the ETH Robotics Club, I collaborated with a multidisciplinary team to develop advanced human–robot interaction capabilities on the Unitree G1 humanoid platform.

We designed and integrated a set of interactive behaviors, enabling the robot to:

- follow a human in real time,
- interpret and respond to hand gestures,
- execute natural language commands.

These functionalities were built on top of a decoupled whole-body control framework (GROOT), allowing for modular and robust integration of perception, planning, and motion control.

In addition, we explored creative AI applications by implementing motion-to-video generation using the GENMO model, linking physical robot actions with generative visual outputs.

This project combined rapid prototyping, robotics, and AI in a highly collaborative environment, emphasizing efficient system integration and real-time problem solving.
